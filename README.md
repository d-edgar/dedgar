## Hosting

This site is served via [GitHub Pages](https://pages.github.com/) from the `docs/` folder. To enable it:

1. Go to **Settings → Pages** in the repo
2. Under **Source**, select **Deploy from a branch**
3. Choose the `main` branch and set the folder to `/docs`
4. Save — the site will be live at `https://d-edgar.github.io/dedgar/`
